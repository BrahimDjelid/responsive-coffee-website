# ☕ Coffee Website

A responsive, single-page website for a coffee shop, built with plain HTML, CSS, and JavaScript.

**Live demo:** https://brahimdjelid.github.io/responsive-coffee-website/

## Features

- Fully responsive layout (desktop, tablet, mobile)
- Animated mobile navigation menu with slide-in panel and dark overlay
- Hero, About, Menu, Testimonials, Gallery, and Contact sections
- Testimonials slider powered by [Swiper.js](https://swiperjs.com/)
- Icons via [Font Awesome](https://fontawesome.com/)
- Contact form UI (front-end only, no backend submission)

## Tech Stack

- HTML5
- CSS3 (custom properties / CSS variables, Flexbox, Grid)
- Vanilla JavaScript
- Swiper.js (CDN)
- Font Awesome (CDN)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/brahimdjelid/responsive-coffee-website.git
   ```
2. Open `index.html` in your browser — no build step or dependencies required.
